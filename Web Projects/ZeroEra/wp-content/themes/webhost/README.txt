== Description ==

WebHost is a WordPress theme especially designed to fulfill the needs of web hosting companies. This theme is Easy to customize and can bring beauty to your business website interface. WebHost comes with some exciting features to create a awesome website. It has the customizing options like blog style, logo update, layout control, color modification, extra widgets, sticky menu and much more.
