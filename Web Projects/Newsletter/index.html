Things to know before joining

(Prerequisites and dependencies)

1. Make Sure To Have At Least The Basic Fundamentals Of CS

Make sure to at least know a small bit of Computer Science. Examples include Using the File Manager, know how to use Microsoft Office and how to use context menus. So basically you have to be comfortable using a computer in general.

2. Be Willing to do the Work

A key component in this group. You have to be Willing To Do The Work. We cannot, absolutely cannot have slackers in this group. If you want to join, great; but if you aren't going to do the work, too bad too sad now fuck off. Simply put, if you want to join the group you have to do the work.

3. Have a Laptop or PC readily available with an active Internet Connection

Not a phone, not a tablet but a laptop or PC. Screen size matters guys, and without a screen size or compatibility with Windows apps, this won't work. Common sense though. Internet really matters too, without it, you'd not be able to do mostly anything at all. Anyway, make sure you have this.

4. Have Fun!

After all that. Have fun! Make sure to check out ThisSite for more info.
